Hexo的基本使用

Hexo
  • Hexo

快速、简洁且高效的博客框架Hexo中文网址：https://hexo.io/zh-cn/

一些常用命令：
  • hexo new "postName" #新建文章
  • hexo new page "pageName" #新建页面
  • hexo generate #生成静态页面至public目录
  • hexo server #启动服务器,预览访问（默认端口4000，’ctrl + c’关闭server）
  • hexo deploy #部署网站(将.deploy目录部署到GitHub)
  • hexo clean #清除缓存文件 (db.json) 和已生成的静态文件 (public)。
  • hexo help # 查看帮助
  • hexo version #查看Hexo的版本
